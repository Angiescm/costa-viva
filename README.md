# 🌴 Costa Viva - Pasaporte Turístico Inteligente

Una plataforma web que conecta turistas con negocios locales mediante un pasaporte interactivo que recompensa cada visita.

## 📋 Descripción del Proyecto

Costa Viva combina:
- 🎫 **Pasaporte Físico** - El turista recibe un pasaporte impreso
- 📱 **Códigos QR** - Cada negocio tiene su QR único
- 🌐 **Plataforma Web** - Consulta información y acumula progreso
- 🎮 **Gamificación** - Sellos, misiones, recompensas
- 📊 **Panel para Negocios** - Estadísticas y gestión de promociones

## 🎯 Características Principales

### Para Turistas
- ✅ Registro de pasaporte (opcional)
- ✅ Dashboard personal (nivel, sellos, puntos)
- ✅ Rutas turísticas temáticas (5 rutas)
- ✅ Mapa interactivo con negocios
- ✅ Escaneo de QR con gamificación
- ✅ Sistema de sellos y recompensas
- ✅ Misiones diarias
- ✅ Comunidad (fotos, calificaciones, recomendaciones)
- ✅ "Explora Cerca de Ti" (basado en ubicación)

### Para Negocios (MIPYMES)
- ✅ Panel de control con métricas
- ✅ Edición de información del negocio
- ✅ Gestión de promociones
- ✅ Estadísticas y analítica
- ✅ Validación de sellos

## 📁 Estructura del Proyecto

```
costa-viva/
├── frontend/              # React + TypeScript + Vite
├── backend/               # Node.js + Express
├── docs/                  # Documentación
└── README.md
```

## 🚀 Stack Tecnológico

### Frontend
- React 18 + TypeScript
- Vite (bundler rápido)
- TailwindCSS (estilos)
- Google Maps API
- QRCode.js

### Backend
- Node.js + Express
- TypeScript
- PostgreSQL
- JWT (autenticación)
- Prisma (ORM)

## 📖 Cómo Empezar

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
npm install
npm run dev
```
